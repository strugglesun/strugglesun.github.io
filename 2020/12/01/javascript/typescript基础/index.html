1.三步设置VScode自动将ts文件编译为js文件

  • 运行tsc –init,创建tsconfig.json文件
  • 修改tsconfig.json文件，设置js文件夹:”ourDir”:”/js/“
  • 设置vscode监视任务，之后修改项目中的ts,自动生成js

3.Typescript 数据类型|元组

  • 概念： 就是一个规定了元素数量和每个元素类型的数组，而每个元素的类型，可以不相同。

  • 语法：

    ​ let 元组名：[类型1，类型2，类型3] = [值1，值2，值3]

  • 为什么要有元组？

    TS数组中元素类型必须一致，如需要不同元素，可以用元组了！

  • 特点：

    ​ 声明时，要指定元素个数

    ​ 声明时，要为每个元素规定类型

11.Typescript函数|可选参数

  • 函数 可选参数

  • 调用

    可以不传实参 函数名();

    可以传递实参 函数名(实参值)；

12.Typescript函数|默认值

  • 函数 默认值(带默认值的参数，本身也是可选参数)

  • 调用

    不传递实参 函数名(); 编译后 函数名(默认值1，默认值2)；

    传1个实参 函数名(实参1)； 函数名(实参1，默认值2)；

    传2个实参 函数名(实参1，实参2)； 函数名(实参1，实参2)；

    只传递2个实参 函数名(undefined,实参2)； 函数名(默认值1，实参2)；

13.Typescript函数|剩余参数

  • 函数 剩余参数

  • 特点

    剩余参数 只能定义一个

    剩余参数 只能定义为数组

    剩余参数 只能定义在形参列表最后

  • 调用

    实参最后的参数都是剩余参数
